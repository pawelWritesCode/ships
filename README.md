# Ships

## Installation:
```bash
    cd frontend
    yarn install
    
    cd ../backend
    yarn install
    yarn create-config
```
Next move to config dir and fulfill dev.js config file.

## Run project in dev mode:
```bash
    cd backend
    yarn dev
```

### API-doc
```bash
    cd backend
    yarn api
```

### fixtures
Need to create your own load-fixtures.sh script first.
```bash
    cd backend
    yarn fixtures
```
